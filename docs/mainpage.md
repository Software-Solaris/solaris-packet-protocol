    <h3 class="spp-section-title">Development roadmap</h3>

    <p class="spp-main-text">
    Version 0.1 & 0.2 – Independent driver implementations for the BMP390 and ICM-20948, supporting continuous sensor data acquisition (completed).<br>
    Version 1.0 – Both sensors integrated into a single general driver (almost ready).<br>
    Version 2.0 – General code running on FreeRTOS (currently on hold).<br>
    Version 3.0 – General code using the Solaris Packet Protocol (under active development).
    </p>

    <h3 class="spp-section-title">Key Features</h3>

    <p class="spp-main-text">
    Engineered for <strong>embedded C</strong> on ESP32-class microcontrollers.<br>
    Modular architecture with a clear separation between <strong>HAL</strong>, <strong>OSAL</strong> and <strong>Core Protocol</strong> layers.<br>
    Inspired by <strong>CCSDS</strong> and <strong>CubeSat Space Protocol (CSP)</strong>, streamlined for high-reliability student rocketry.<br>
    Fully <strong>open-source</strong>, designed for both <strong>education</strong> and <strong>flight-ready deployments</strong>.
    </p>
